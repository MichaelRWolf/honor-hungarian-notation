# Honor Hungarian Notation

> "I come to honor Hungarian Notation, not to mock it."
>
> — _for Hungarian Notation is an honorable practice_

A Shakespearean denunciation of Hungarian Notation, structured as a parallel to
Mark Antony's funeral oration from _Julius Caesar_ (Act III, Scene II).

The speech "defends" Hungarian Notation. Every line is a takedown.

The text is then decorated with Part-of-Speech annotations using the same
convention it mocks — because Hungarian Notation is an honorable practice.

## Interactive Piece

Published at [michaelrwolf.github.io](https://michaelrwolf.github.io/) with four
view modes:

1. **As Written** — the speech as prose; readable without annotation
2. **POS-Tagged** — every word prefixed with its part of speech (`n_`, `v_`, `adj_`)
3. **Tag-Receding** — annotations present but dimmed; prose shines through
4. **Grammar-Lit** — words color-coded by grammatical role

The toggle button is the argument.

## Tooling
